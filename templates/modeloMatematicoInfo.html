Modelo Matematico

¿Que es un Modelo Matematico?

Un modelo matematico es una representación abstracta de un sistema real

Los modelos matematicos son utilizados para describir, explicar y predecir el comportamiento de ciertos fenomenos del mundo real.

Modelo SIR

El modelo SIR es un modelo matemático hecho para describir la propagación de enfermedades infecciosas en una población, para el desarrollo de este proyecto se adapto nuestras variables al modelo SIR.

Se creó un sistema de ecuaciones diferenciales con 5 variables.

Metodo RungeKutta

El metodo RungeKutta es una familia de metodos numericos utilizados para resolver ecuaciones diferenciales ordinarias, para el desarrollo de este proyecto utilizamos el metodo de RungeKutta Orden 4.

El sistema de ecuaciones diferenciales se resolvio através de este metodo numerico
